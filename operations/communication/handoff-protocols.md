# Handoff Protocols

## Designer -> Frontend
- Tasarım konsepti ve renk paleti netleştirilmeden kodlamaya başlanmaz.
- Assets ve ikonlar belirlenen klasörlere aktarılır.

## Frontend -> Backend
- API gereksinimleri (Endpoint, Request/Response yapısı) önceden belirlenir.
- Mock veriler ile geliştirme yapılabilir ancak entegrasyon için Backend onayı şarttır.

## Backend -> Frontend
- API dokümantasyonu veya şeması sağlanır.
- Hata kodları ve veri formatları standartlara uygun iletilir.
